Salmon Run Wild


Salmon Run Wild is a single player side-scrolling game written in JavaScript using the p5.js creative coding library. Participants play as a spawning salmon, running upstream with the goal of fertilizing eggs while avoiding predators. The game was developed for a 2018 BYOB event at VIVO media center in Vancouver BC and currently exists as an early-stage proof-of-concept. Several aspects of the gameplay, programming, and design are still unfinished, and will continue to be developed in the future.


The core design of Salmon Run Wild originated from a frustration with the popular mobile game (now defunct) Flappy Bird. Based on the 2000 original Helicopter Game, Flappy Bird gained viral success in 2014, but lacks the playability of its precursor due to its unimaginative physics simulation. Both games use a single-button input to maneuver a flying object around obstacles, however, where Helicopter Game feels fast, dynamic, and dangerous, Flappy Bird feels slow and unresponsive. Salmon Run Wild is intended to recapture the joy that Helicopter Game succeeded in creating through inventive and subtle physical mechanics in single-button gameplay.
